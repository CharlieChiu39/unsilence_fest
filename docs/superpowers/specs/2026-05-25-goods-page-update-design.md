## 視覺風格

**A 方案（反差萌）**：商品圖原樣放入 Win95/Cyberpunk 卡片中。
- 圖片區白底（`#ffffff`），讓彩色 pop 商品圖在暗色 OS 視窗裡跳出
- 卡片背景維持深紫黑（`rgba(10, 5, 20, 0.85)`）
- 邊框：Win95 凸起樣式（`border-top/left: #fff, border-right/bottom: #555`）
- hover：輕微 glitch shake + 邊框換 neon 主色
